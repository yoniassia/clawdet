🏗️ MoneyClaw V1 — 1,000 Instances (Current Container Topology)

Existing production architecture + the best cost/perf Hetzner sizing to reach ~1,000 OpenClaw instances with headroom.

✅ Existing Topology (Today)

  • 3-node cluster: main + worker1 + worker2 (WireGuard 10.10.0.1/2/3)
  • Main: provisioner, auth-proxy, Redis, Postgres, LiteLLM, nginx
  • Workers: OpenClaw tenant containers + worker-api
  • Per-tenant containers: 1GB limit, ~330–356MB avg
  • Gateway heap: 768MB minimum for stability
Main (control plane)
  ├─ Provisioner
  ├─ Auth-Proxy
  ├─ Redis / Postgres
  ├─ LiteLLM proxy
  └─ Nginx + DNS
Workers (agent containers)
  ├─ worker1: tenant containers
  └─ worker2: tenant containers

💡 Hetzner V1 Plan (Best Perf/Cost)

  • Control plane (HA): 2× AX52 (Ryzen 7700, 64GB)
  • Workers: 10× AX102 (Ryzen 7950X, 128GB)
  • Capacity: ~100 instances per worker with 30–40% headroom
  • Network: Hetzner private network + NVMe on all nodes
  • Monthly range: ~$2.6k–$3.0k total (control + workers)

Why this mix?

AX102 gives the best CPU/RAM/$ for agent containers. Two smaller control nodes keep the gateway + provisioner stable under spikes while workers scale horizontally.

💡 Cost Insight

Infrastructure is <5% of total cost at scale — LLM API costs dominate. This is why Hetzner's 5-10x savings on infra matters less at 500K than smart LLM routing. Invest in the Quant Router approach for real savings.

🗺️ Migration Roadmap

📍 Current — Single Server

1 Hetzner VPS, Docker Compose, OpenClaw, <100 agents. Cost: ~$15/mo.

🎯 Phase 1 — 5K Agents

2-3 Hetzner nodes, K3s cluster, Cilium network policies, hot/cold pools. Cost: ~$150/mo.

🚀 Phase 2 — 50K Agents

8-16 Hetzner nodes, multi-zone K3s, KEDA auto-scaling, Falco monitoring, dedicated DB nodes. Cost: ~$2K/mo.

🌍 Phase 3 — 500K Agents

64-128 nodes, multi-region, custom provisioner v2, potential IronClaw for enterprise tier, dedicated security team. Cost: ~$20K/mo.

⚠️ Key Milestones

  • At 1K agents: Must move from Docker Compose to K3s
  • At 10K agents: Must implement hot/cold architecture
  • At 50K agents: Must add multi-zone for availability
  • At 100K agents: Consider dedicated enterprise tier (IronClaw)

🏆 MoneyClaw Differentiation

1. Cost Leadership

Hetzner + K3s = 5-10x cheaper than competitors on AWS/Azure. We pass savings to customers or keep as margin.

2. Ecosystem Breadth

OpenClaw's 11+ platforms and 5,700+ skills give us instant capability that competitors building on leaner claws lack.

3. Security Layering

We compensate for OpenClaw's app-level security with infra-level hardening: K3s namespaces, Cilium, Podman, Falco.

4. Smart LLM Routing

Quant Router auto-classifies queries and routes to cheapest capable model. 94% cost savings vs premium-only.

🔮 Ecosystem Gaps & Opportunities

1. Multi-Agent Collaboration

Primitive across all claws. First to crack agent-to-agent cooperation wins.

2. Agent Testing Framework

No claw has proper testing tools. Huge opportunity for open-source contribution.

3. Observability & Debugging

Afterthought everywhere. Need agent-specific tracing and debugging tools.

4. Skill Quality & Safety

ClawHub has 5,700+ skills but no quality gate. Automated verification needed.

5. Non-Developer Onramp

All claws require technical setup. Consumer-friendly onboarding = massive TAM unlock.

6. Cross-Claw Skill Portability

No universal skill format. MCP is closest but incomplete. Standard needed.
